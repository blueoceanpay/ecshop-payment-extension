## Install (適用於 Linux)
1. 進入 ecshop 安裝根目執行：git clone https://github.com/blueoceanpay/ecshop-payment-extension.git
2. cd ecshop-payment-extension
3. 執行　./install.sh
4. 安裝完畢！
